# Operations Intelligence Copilot

Operations Intelligence Copilot is an AI-powered incident management and analytics platform designed to help IT Operations teams manage incidents, generate AI-driven insights, monitor operational health, and automate stakeholder reporting.

Developed as a Capstone Project to demonstrate AI-powered incident management, operational analytics, and automated stakeholder reporting using modern cloud-native technologies.
